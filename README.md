# Radio Dream Voice

## Overview
Radio Dream Voice is an online radio station project dedicated to providing a platform for diverse musical experiences. Our goal is to bring together music lovers from all around the world, offering a wide variety of genres and live shows.

## Features
- **Live Streaming:** Enjoy live broadcasts from our studio, featuring various music genres and special guests.
- **On-Demand Listening:** Access recorded shows and playlists at your convenience.
- **User Interaction:** Engage with the station through chat, requests, and social media.

## Getting Started
To get started with the Radio Dream Voice project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/radiodream2026/radio-dream-voice.git
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
